# Etch-a-Sketch

A simple drawing grid where users can draw by hovering over squares.

## Description
This project is part of TOP curriculum Foundations course and focuses on DOM manipulation and event handling.

## Built With
- HTML
- CSS
- JavaScript

## Features
- Adjustable grid size
- Drawing on hover
- Clear button

## What I Learned
- DOM manipulation
- Event listeners
- Flexbox layout
